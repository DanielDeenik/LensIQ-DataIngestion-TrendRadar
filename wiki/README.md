## Adding Screenshots

To add screenshots to the wiki:

1. **Capture Screenshots**
   - Use high-resolution screenshots (1920x1080 or higher)
   - Ensure UI is clean and professional
   - Hide sensitive data

2. **Optimize Images**
   - Save as PNG for UI screenshots
   - Save as JPG for photos
   - Compress images to reduce file size
   - Recommended max size: 500KB per image

3. **Name Images**
   - Use descriptive, lowercase names
   - Use hyphens for spaces
   - Example: `strategy-hub-dashboard.png`

4. **Add to Wiki**
   - Place images in `wiki/images/` directory
   - Reference in markdown: `![Alt Text](images/filename.png)`
   - Add descriptive alt text for accessibility

See [Screenshot-Guide.md](Screenshot-Guide.md) for detailed instructions.
